# BELIER 통합 업무 매뉴얼 — 마케팅

벨리에 마케팅팀 업무 매뉴얼. 기획 → 생산 → 물류 → 마케팅 → 판매 → CS 전체 flow 중 마케팅 파트 담당.

## 구조
- 단일 HTML (`index.html`) — 프레임워크 없음, 브라우저만 있으면 열림
- 9개 탭: 회사 전체 flow · 업무 카테고리 맵 · 일일 루틴 · 프로세스 시트 (8) · 외부 협업 · 도구 가이드 · 월간 정산 · 용어집 · 템플릿

## 기능
- **편집 모드** (주황 버튼) → 텍스트 인라인 수정
- **저장** → 브라우저 localStorage에 저장 (본인 브라우저에만 유지)
- **HTML 내보내기** → 수정된 상태를 .html 파일로 다운로드 (다른 사람과 공유)
- **복사** → 템플릿 멘트 즉시 클립보드

## 배포
Vercel 자동 배포 (main 브랜치 push → 배포).
