💡 마케팅의 역할
생산된 상품이 고객에게 "왜 필요한지"를 전달하고, 판매로 이어지게 하는 단계입니다. 위로는 기획/생산의 스토리를 받고, 아래로는 CS/재구매 데이터를 피드백합니다.

📥 위에서 받는 것 (Input)

  • • 시즌 발매 일정 (기획팀)
  • • 라인시트 · 샘플 (기획/디자인팀)
  • • 생산 완료 일정 (생산팀)
  • • 대표 승인 (예산·행사·콜라보)

📤 아래로 넘기는 것 (Output)

  • • 콘텐츠 (스토리·피드·릴스·룩북)
  • • 광고/CRM 캠페인 노출
  • • 협찬/시딩 발송 리스트 → 물류
  • • 판매 리포트 · ROAS → 대표/기획

🔄 반복 주기

  • • 일간: 스토리·채널 운영·CRM
  • • 주간: 기획 회의·성과 리포트
  • • 월간: 비용 정산·협업 정산
  • • 시즌: 발매 행사·룩북·대형 캠페인

용어집

신규 담당자 첫 1주일 필수. PPT 곳곳에 나오는 내부 용어 정리. 이해 안 되는 단어는 편집 모드에서 추가.

🛠 툴·플랫폼

데이터라이즈
벨리에가 쓰는 CRM·온사이트 마케팅 툴. 온사이트 캠페인(자사몰에 팝업·배너 노출) + 메시지 캠페인(이메일·문자·앱 푸시)을 함께 운영. 월별 사용 내역서가 메일로 와서 비용 정산의 주요 항목.
어치브먼트
연예인·셀럽 협찬 요청 플랫폼. 여기로 오는 협찬은 반납 기반이라 협찬서 작성 필수. 매일 아침 메일과 함께 확인.
이지어드민
내부 발주 관리 시스템. 협찬·시딩 발송 시 수동발주로 처리. 급건은 퀵배차 연계.
채널톡
고객 상담 메신저. CS팀이 주로 쓰지만 마케팅은 초과 사용량 중 마케팅 발송분(문자·소셜미디어)을 월별 비용 정산에 반영.
크리마
리뷰 수집·적립금 지급 툴. 리뷰 발췌해 마케팅 소재로 사용 가능.

📋 시트·문서

기획시트 · 라인시트
시즌별 상품 마스터 Google Sheets. 샘플 위치·재고·이미지·발매일 확인. 협찬 샘플 찾을 때 가장 먼저 보는 시트.
브랜드 운영 시트
벨리에 운영 전반 Google Sheets. 개별_브랜드이슈 탭이 마케팅 비용 정산·협찬 성과 입력의 핵심.
마케팅 리스트
신상품 발매 시 플랫폼별 등록 체크리스트. 특히 '체크리스트 탭' 확인.
협찬서
벨리에 제품 협찬 시 동봉하는 양식. 연예인 반납 기반 협찬 시에만 작성 (일반 무상 협찬은 불필요).

👕 벨리에 내부 용어

크루 (Crew)
벨리에 회원. 회원 등급은 크루컬러로 표현 — White → Blue → Navy → Purple → Black.
크루컬러 / 컬러쿠폰
등급별 할인 쿠폰. 매월 1일 자동 갱신. Black 최초 달성 시 케어키트 증정.
크루데이
특정 컬러 등급 대상 할인 이벤트. 데이터라이즈 캠페인과 연계.
테이크아웃 브랜드
벨리에 여성 카테고리 시딩 파트너사. 주간 단톡방 소통 + 신상 발매 시 셀렉 리스트 공유.
플래그십스토어
벨리에 주력 오프라인 매장 (서울숲·롯데월드몰·롯데본점·명동본점·더현대서울). 시즌 전환 시 매장 사진 변경 필수.

📊 마케팅 지표

ROAS
Return On Ad Spend. 광고비 1원당 매출 얼마. (ROAS 300% = 광고비 1만원 쓰면 매출 3만원)
CTR
Click-Through Rate. 노출 대비 클릭 비율. (CTR 2% = 노출 100번 중 2번 클릭)
CAC
Customer Acquisition Cost. 신규 고객 1명 획득에 들어간 마케팅 비용.
시딩 (Seeding)
인플루언서/셀럽에게 제품 무상 제공 → 자연스러운 착용·노출 유도. 반납 없음, 협찬서 불필요.
리그램
인스타 스토리에서 다른 계정 게시물 재공유. 행사 중 인플루언서 스토리 리그램이 핵심 바이럴.

👥 사람·역할

MD
Merchandiser / 마케팅 담당자. 이 매뉴얼의 주 사용자. PPT에서 "MD 인수인계서"는 박준홍님이 작성.
정다은님
피드 기획 협업 (외부/내부 확인 필요). 주 2~3회 피드 기획 미팅.
승환님
총괄 · 생산팀. 교환/반품 승인 · 협찬 예산 컨펌.
찬웅님
생산팀 (1년차, 정형 아이템). AS 판단.
경빈님
경영지원. 무통장 환불 · 커피쿠폰 발송 등 처리.

  • ☐ 대표 컨펌 완료

  • ☐ 동봉물: 협찬서 + 감사 카드 + (해당 시) 부자재
