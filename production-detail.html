주요 문제점 & 해결 방안

문제 1: 작지 부자재 리스트 오류

현상: 디자이너가 작지에 부자재를 기입할 때 잘못된 정보(업체명 누락, 수량 오류, 중복 기입 등)를 넣음. 생산팀이 이를 검증 없이 발주하면 잘못된 부자재가 입고됨.

원인: 디자이너와 생산팀 사이에 부자재 리스트 검증 단계가 없음. 앱스크립트가 작지 데이터를 "있는 그대로" 가져오므로 오류도 그대로 반영됨.

해결:
단계액션담당
즉시작지 부자재 검증 체크리스트 도입 (7탭 참조). 찬웅이 작지 링크 입력 전에 반드시 확인찬웅
1주 내handleWorkOrderLink()에 검증 로직 추가: 업체정보 누락/수량 0/품목명 비어있음 → 경고 메시지앱스크립트
2주 내디자이너에게 부자재 기입 가이드 공유, 필수 필드 정의승환

문제 2: 중간 추가 아이템 누락

현상: 시즌 중간에 추가되는 스타일이 제품리스트에는 들어가지만 생산현황에 반영되지 않아 관리 사각지대 발생.

원인: 제품리스트 → 생산현황 이관이 수동. 바쁜 시기에 잊어버림.

해결:
단계액션담당
즉시매주 월요일 "신규 품번 체크" 루틴 — 제품리스트 vs 생산현황 품번 대조찬웅
2주 내앱스크립트로 "미등록 품번 감지" 기능 추가 — 제품리스트에 있는데 생산현황에 없는 품번 자동 리스트업앱스크립트
1개월제품리스트 등록 시 생산현황에 자동 행 추가 (반자동)앱스크립트

문제 3: 케어라벨 일괄 발주 불가

현상: 케어라벨은 스타일별로 발주하지 않고 공장 투입 시점에 맞춰 일괄 발주해야 하는데, 전체 수량이 확정되지 않으면 발주할 수 없음. 중간에 추가되는 스타일 때문에 더 복잡해짐.

원인: 케어라벨 수량 = 전체 확정 수량이 필요한데, 스타일이 순차적으로 확정되므로 "전체"를 잡기 어려움.

해결:
단계액션담당
즉시케어라벨 발주를 "차수별 일괄"로 변경 — 해당 차수 기획량 확정 시 일괄 발주승환
1주 내생산현황에서 차수별/혼용율별 케어라벨 필요 수량 자동 합산 시트 추가앱스크립트
2주 내여유분(+3~5%) 포함한 자동 계산, 발주 시점 알림앱스크립트

문제 4: 출고예정일 미기입 (관리 사각지대)

현상: 투입은 했는데 출고예정일(AK열)이 비어있는 건이 20건 이상. 납기 관리 불가.

해결:
단계액션
즉시투입 체크(AL열) 완료 후 AK열 비어있으면 → 배경색 빨간색으로 경고
1주 내빠른 갱신에 "미기입 경고" 추가 — 투입완료인데 AK/AJ 비어있는 행 자동 하이라이트

문제 5: N8N 발주서 — 업체정보 누락 시 미생성

현상: findAllOrderItems()가 "본사발주 + 업체정보 있는 것만" 필터링 → 업체정보 누락된 부자재는 발주서가 안 만들어짐.

해결:
단계액션
즉시T7 체크 전에 "업체정보 미입력 항목" 경고 팝업 표시 (앱스크립트)
2주 내N8N에서 누락 항목 별도 시트에 기록 → "발주 누락 리스트" 자동 생성

  • 원단정보(원단처, 원단명, 컬러, 혼용율, 원단폭, 용척)가 모두 있는가?
  • 후가공 여부가 표시되어 있는가?

Gate B — 부자재 리스트 검증

  • 자재발주 시트에 부자재 항목이 하나라도 있는가? (빈 시트 아닌지)
  • 각 부자재의 업체정보가 모두 입력되어 있는가? (빈칸 없는지)
  • 각 부자재의 품목명이 정확한가? (오타, 중복 없는지)
  • 수량이 0이거나 비정상적으로 큰 항목이 없는가?
  • 구분 열에 "부자재"와 "공용부자재"가 올바르게 분류되어 있는가?
  • 택/라벨/케어라벨이 부자재 리스트에 포함되어 있는가?
  • 동일 부자재가 중복 기입되지 않았는가?

Gate C — 발주 전 최종 확인

  • 작지의 사이즈/수량과 제품리스트의 기획수량이 일치하는가?
  • "본사발주" 항목과 "업체발주" 항목이 올바르게 구분되어 있는가?
  • 기존에 같은 스타일의 발주서가 이미 생성되지 않았는가? (중복 방지)
  • 모든 필수 정보가 채워진 후 T7 체크할 준비가 되었는가?
체크리스트 활용법:
찬웅이 작지를 받으면 → Gate A 확인 → Gate B 확인 → 문제 없으면 생산현황 I열에 작지 링크 입력 → 자동 추출 실행 → AG/AH열 결과 확인 → Gate C 확인 → T7 체크 (발주서 생성)

투입 전 체크리스트

Gate 2 — 공장 투입 전 필수 확인

  • 원단 공장 입고 확인됨? → AF열 = "입고완료"
  • 부자재 전량 사무실 입고 확인됨? → AG열 = 100%
  • 공용부자재 확인됨? → AH열 = 100%
  • 부자재 키트 포장 완료됨? → BH열 확인
  • 택/라벨/케어라벨 포함됨?
  • 작지(작업지시서) 최종 검토 완료?
  • 투입예정일(AJ열) 기입됨?
  • 출고예정일(AK열) 기입됨?
